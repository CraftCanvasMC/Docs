Canvas is a fork of Folia that aims to provide a stable, fast, and efficient alternative to Folia. The recent versions of Folia have consistently presented
instability issues, lack of maintenance, and incomplete API support for plugins to work with region threading. The Canvas project is an attempt to fix these
shortcomings by focusing on reliability, quality, performance, and long-term maintainability.

The project is actively maintained by a team of developers who understand Folia and its inner workings at a deep level.

Canvas aims to:

- Provide a comprehensive and powerful API to assist plugins in supporting region threading.
- Maintain active development with a dedicated team
- Deliver high quality and stable optimizations and features for server owners.
- Provide fast and reliable support for issues and feature requests

## Contributing

We welcome contributions in many forms:

- **Code**: Submit pull requests for bug fixes and features
- **Documentation**: Help improve our guides and API documentation
- **Testing**: Report bugs and test new features
- **Community**: Help other users and share your experience
- **Donating**: Help the Canvas team keep Canvas' services running

Visit our [Contributing Guide for Canvas](developers/contributing/canvas) or the [Contributing Guide for Docs](developers/contributing/docs) to get started.
