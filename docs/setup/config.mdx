CanvasMC offers a robust set of configurable options that let you fine-tune server performance and behavior. The configuration parameters are organized into logical sections for easy reference.

<Callout type='info' title='Optimization Notice'>
  The default configuration settings are optimized for a balance between performance and stability. Adjust these
  parameters as needed to best fit your server's hardware and workload.
</Callout>

## General Settings

| Setting                               | Description                                                           | Default |
| ------------------------------------- | --------------------------------------------------------------------- | ------- |
| `runPlayerListTickOnIndependentLevel` | Runs player list ticks on independent level threads                   | `true`  |
| `timeResyncInterval`                  | Number of ticks between time resynchronization events per world       | `400`   |
| `levelThreadPriority`                 | Thread priority for level threads                                     | `9`     |
| `useLevelThreadsAsChunkSourceMain`    | Directs ServerChunkCache tasks to level thread instead of main thread | `true`  |
| `emptySleepPerWorlds`                 | Enables per-world "empty server" sleep logic                          | `true`  |
| `enableCanvasTickCommand`             | Activates the `threadedtick` command                                  | `true`  |

## Chunk Generation Options

| Setting                                     | Description                                        | Default      |
| ------------------------------------------- | -------------------------------------------------- | ------------ |
| `chunkGeneration.chunkDataCacheSoftLimit`   | Soft limit for chunk data caching                  | `8192`       |
| `chunkGeneration.chunkDataCacheLimit`       | Hard limit for chunk data caching                  | `32678`      |
| `chunkGeneration.allowAVX512`               | Enables AVX512 instruction support when available  | `false`      |
| `chunkGeneration.nativeAccelerationEnabled` | Activates native acceleration for chunk generation | `true`       |
| `chunkGeneration.chunkWorkerAlgorithm`      | Algorithm for thread count determination           | `"moonrise"` |

## Pathfinding Settings

| Setting                                     | Description                       | Default |
| ------------------------------------------- | --------------------------------- | ------- |
| `pathfinding.enableThreadedPathfinding`     | Enables asynchronous pathfinding  | `true`  |
| `pathfinding.useThreadedWorldForScheduling` | Uses dedicated threaded scheduler | `true`  |
| `pathfinding.maxProcessors`                 | Maximum cores for pathfinding     | `2`     |
| `pathfinding.keepAlive`                     | Idle thread lifetime in seconds   | `60`    |

## Entity Tracking Settings

| Setting                                 | Description                        | Default |
| --------------------------------------- | ---------------------------------- | ------- |
| `entityTracking.enableThreadedTracking` | Shifts tracking to separate thread | `true`  |
| `entityTracking.maxProcessors`          | Maximum cores for tracking         | `1`     |
| `entityTracking.keepAlive`              | Idle thread lifetime in seconds    | `60`    |

## Performance Settings

| Setting                           | Description                                     | Default |
| --------------------------------- | ----------------------------------------------- | ------- |
| `enableAsyncSpawning`             | Activates asynchronous mob spawning             | `true`  |
| `skipTicksAdvancements`           | Delay between inventory triggers                | `3`     |
| `skipUselessSecondaryPoiSensor`   | Disables unnecessary POI sensor ticking         | `true`  |
| `optimizePistonMovingBlockEntity` | Optimizes piston block entities                 | `true`  |
| `clumpOrbs`                       | Groups XP orbs for efficiency                   | `true`  |
| `useCompactSineLUT`               | Uses compact lookup table for trig calculations | `true`  |

## TNT Settings

| Setting                           | Description                          | Default |
| --------------------------------- | ------------------------------------ | ------- |
| `tnt.enableFasterTntOptimization` | Enables TNT performance enhancements | `true`  |
| `tnt.explosionNoBlockDamage`      | Disables block damage from TNT       | `false` |
| `tnt.tntRandomRange`              | Random range modifier for TNT        | `-1.0`  |

## Summon Command Settings

| Setting                   | Description                      | Default |
| ------------------------- | -------------------------------- | ------- |
| `summonCommandBatchCount` | Entities to summon per tick      | `50`    |
| `batchSummonCommandTasks` | Groups summon tasks across ticks | `true`  |

## Lag Compensation Settings

| Setting                                       | Description                         | Default |
| --------------------------------------------- | ----------------------------------- | ------- |
| `lagCompensation.enabled`                     | Master switch for lag compensation  | `true`  |
| `lagCompensation.blockEntityAcceleration`     | Speeds up block entity handling     | `false` |
| `lagCompensation.blockBreakingAcceleration`   | Accelerates block breaking          | `true`  |
| `lagCompensation.eatingAcceleration`          | Improves eating response time       | `true`  |
| `lagCompensation.potionEffectAcceleration`    | Enhances potion effect speed        | `true`  |
| `lagCompensation.fluidAcceleration`           | Optimizes fluid updates             | `true`  |
| `lagCompensation.pickupAcceleration`          | Improves item pickup response       | `true`  |
| `lagCompensation.portalAcceleration`          | Optimizes portal transitions        | `true`  |
| `lagCompensation.sleepingAcceleration`        | Reduces sleep delays                | `true`  |
| `lagCompensation.timeAcceleration`            | Adjusts time progression during lag | `true`  |
| `lagCompensation.randomTickSpeedAcceleration` | Boosts random tick performance      | `true`  |

## Advanced Entity Settings

| Setting                    | Description                              | Default |
| -------------------------- | ---------------------------------------- | ------- |
| `entityMasks`              | Configurable tick masks for entities     | `[]`    |
| `threadedEntityTicking`    | Enables async entity ticking             | `false` |
| `disableEntityPushing`     | Disables non-player entity pushing       | `false` |
| `alwaysAllowWeirdMovement` | Prevents movement corrections during lag | `true`  |
